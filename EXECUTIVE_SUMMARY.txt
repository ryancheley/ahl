KEY DISCOVERIES
================================================================================

1. /feed Endpoint Exists But Non-Functional
   • Returns "Invalid key" or "Unsupported feed"

   • No supported feed types found
   • Acts as access control gate with no valid credentials

2. No REST API Infrastructure
   • /api/v1/, /api/v2/, /api/v3/ - all 404
   • /api/games/ - 404
   • /api/game/{id} - 404
   • Pattern is consistent: REST not implemented

3. Only HTML Game Reports Work
   • /game_reports/official-game-report.php - 200 OK ✓
   • Contains all game data in HTML format ✓
   • This is the ONLY reliable source for detailed data

4. Query Parameters Have No Effect
   • &format=json - Ignored (still returns HTML)
   • &api=true - Ignored
   • &details=full - Ignored
   • Adding parameters does not enable JSON mode

5. api.theahl.com Is Restricted
   • Returns 403 Forbidden
   • Likely authentication-required
   • Not accessible to public users
